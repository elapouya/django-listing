Do not forget to declare django_listing css in your header :

<link rel="stylesheet" href="{% templatetag openblock %} static "django_listing/css/django_listing.css" {% templatetag closeblock %}">

Then you can display an icon in your templates by adding :

<span class="the_icon_name"></span>

If you want to animate the icon (spinner), add the animate-spin class :

<span class="listing-icon-spin6 animate-spin"></span>
