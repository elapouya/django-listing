Reference guide : App config

Introduction

You can configure django-listing in settings.py by using this syntax : 

{% codelisting %}DJANGO_LISTING = {
    "PARAMETER1": "value1",
    "PARAMETER2": "value2",
    ...
}{% endcodelisting %}


Parameters

NameDescriptionDefault value
AUTO_DECLARE_CSSIf False, developer must declare himself django_listing.min.css, datetimepicker.min.css, dropzone.min.css and other css filesTrue
