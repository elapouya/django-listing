Readme first : Installation

Packages

To install the main package use pip :


    pip install django-listing
If you want to use autocomplete in filters or attached form : 

    pip install django-autocomplete-light

Configuration

in settings.py :


    INSTALLED_APPS = [
        ...
        "django_listing",
        ...
    ]
    ...
    DJANGO_LISTING = {
        "THEME": "bootstrap5",  # Could be default, bootstrap4, bootstrap5
    }
In your django template you must include a django-listing header and footer templatetags as well as bootstrap, jquery and poper if you are using it : 
{% codelisting %}{% verbatim %}
    {% load django_listing %}
    ...
    <html>
    <head>
    ...
    

    {% render_listing_header %}
    ...
    </head>
    <body>
    ...
    {% render_listing  %}
    ...
    
    
    {% render_listing_footer %}
    </body>
    </html>
    {% endverbatim %}{% endcodelisting %}
Note : jquery, popper, bootstrap javascripts are not mandatory for basic listings
Note 2 : django-listing header and footer and css+js declarations are usually put in a django base template
