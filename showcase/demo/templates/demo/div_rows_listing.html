Div listing

Django-listing is able to create listings made only with <div> tags. This is the best way to have rich rows, or several records on the same row (thumbnail listing).

To do that, you must set in your listing class attribute template_name='django_listing/listing_div.html' OR subclass DivListing

Then, each row will be rendered following another template specified in div_template_name attribute. Here, look at demo/templates/demo/div_row.html template

One div by row :

{% render_listing EmployeeDivListing Employee per_page=5 %}

Many divs by row :

You can have many divs/records in a same row, just resize the container so you can put many within the listing width.
Specify a display: inline-block; width=...; height=...; on the container.

In this example, div_template_name='demo/templates/demo/thumbnail.html', the template displays only a picture, first_name, last_name and age.

{% render_listing EmployeeThumbnailsListing Employee per_page=16 %}

NOTE : The images are generated with the django application : django-robohash-svg
